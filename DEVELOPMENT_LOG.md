## 解決した主な課題

### データ取得
- GitHub APIを使用して正しいデータパスを特定
- 都道府県コードの自動生成

### 表示の問題
- 北海道が画面外に出る → スケール調整
- 沖縄県が表示されない → 特別な配置ロジック
- 東京都の離島で本土が選択できない → フィルタリング機能
- 新潟県が消える → 佐渡島は残すように調整

### ユーザビリティ
- ズーム中に他の県に入ってしまう → 自動ズーム機能を削除
- ホバーでビューがリセットされる → 選択時のみビューボックスを更新
- ドラッグ中の誤クリック → ドラッグ中はクリックを無視
